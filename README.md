# G2CTool_project
Graphical Converter from GENEPOP to CERVUS for Population Genetics Analyses

Ce logiciel, développé par Karim Ghazouani et Faiza Chbel, permet la conversion de fichiers GENEPOP vers le format CERVUS utilisé par le logiciel CERVUS en génétique des populations.
Il fonctionne uniquement sur Linux ; Python n’est pas obligatoire mais peut être installé si nécessaire. Pour utiliser ce logiciel, placez le fichier exécutable G2CTool dans un dossier de votre choix. 
Ouvrez un terminal Linux, dirigez-vous vers le dossier contenant le fichier avec cd ~/Desktop (ou cd ~/Bureau selon la langue du système), puis donnez une fois le droit d’exécution au programme avec chmod +x G2CTool.
Ensuite, lancez le programme avec ./G2CTool : un widget graphique s’ouvrira vous permettant de sélectionner votre fichier GENPOP d’entrée, puis de choisir le dossier et le nom du fichier de sortie SERVUS. 
Attention : le fichier GENEPOP doit être formaté correctement, chaque population doit commencer par le mot Pop, et il ne doit pas contenir de titres supplémentaires, seulement les noms des allèles, comme indiqué dans l’exemple fourni.
Pour guider les utilisateurs, un fichier EXAMPLE TEMPLATE est fourni. Après confirmation, la conversion se fera automatiquement et le fichier SERVUS sera créé à l’emplacement choisi. 
Ce logiciel fonctionne en ligne et nécessite obligatoirement une connexion Wi-Fi.
